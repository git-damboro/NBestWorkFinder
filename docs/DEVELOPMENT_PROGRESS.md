## 本地编译方式

| 场景 | 命令 | 说明 |
|---|---|---|
| 后端最小编译 | `$env:JAVA_HOME='G:\jdk'; .\gradlew.bat :app:compileJava` | 后端需要使用 `G:\jdk` 作为 JDK 21 路径 |
| 前端构建 | `cd frontend; npm.cmd run build` | 前端 TypeScript 与 Vite 构建 |
