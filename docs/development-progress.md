## 7. 技术债与风险

| 类型 | 内容 | 当前结论 |
|---|---|---|
| 前端构建告警 | 当前构建存在既有 CSS 压缩告警与大 chunk 告警 | 不影响本轮职位工作台交付，后续可单独做构建优化 |
| 历史数据 | 旧数据中可能存在 `user_id = null` 的知识库 / RAG 记录 | 开发阶段可接受默认不可见，后续视需要补迁移脚本 |
| 测试覆盖 | 本轮以前端构建验证与最小后端单测为主 | 符合当前快速开发节奏，后续逐步补强端到端与权限测试 |
| 跨模块联动 | 目前职位、简历、面试已具备能力，但流程仍偏分散 | 下一阶段优先补齐连续使用链路 |
